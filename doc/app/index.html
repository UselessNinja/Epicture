EPICTURE

Epicture is a mobile application developed in Kotlin to interact with the famous American online image sharing imgur thanks to his API. The goals of this project are:
- Display the pictures online thanks to an account
- To implement possibilities to do research
- To implement possibilities to upload pictures
- Create some favorite handler
- Create filter management
This application was developed to work with SDK API 28 and is developed to be the much user-friendly possible.
